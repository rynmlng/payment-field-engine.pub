# New Generator Guide

This is a guide for how to add a new generator for this site.

## 1. Add A New Page Via Front Matter
Add a new page inheriting from the `payment_field` layout
Add `fields` to generate the form dynamically
Add `howto` properties for the HowTo AEO schema
Add `faq` properties for the FAQ AEO schema

## 2. Add New Assets
Take a screenshot of the form, save it into `images/forms`, and add `how_to.form_image` properties in the Front Matter

## 3. AEO and SEO
Update `ai.txt` and `llms.txt`

## 4. Community
Update `README.md`
