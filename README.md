## Introduction

Payment fields outside the US are confusing, seemingly random, yet surprisingly deterministic. This site will serve as a guide to be able to understand and generate those fields for you. Provide the inputs and the output will unblock you from processing payments. 

## Fields Supported

* Brazil IBAN
* Canada Routing Number
* China CNAPS (Coming Soon)
* Mexico CLABE
